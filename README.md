PenTest Assistant
PenTest Assistant is an AI-powered tool designed to support ethical and authorized penetration testing. It helps users analyze scan results, identify vulnerabilities, and generate professional security reports.
The application is built with Gradio for the user interface and uses the OpenAI API for language model capabilities.
This tool is intended only for legal and authorized use. Users must have explicit permission to test any target system. The application enforces this by limiting sensitive functionality unless authorization is confirmed.
To get started, clone the repository, install the required dependencies, and provide your OpenAI API key through an environment variable or a .env file. After setup, you can run the application locally and access the interface in your browser.
The assistant allows users to interact through a chat interface, upload scan or log files for analysis, and generate structured penetration testing reports based on collected findings.
